Como usar
1. Ative o GPS — o Chrome pedirá permissão de localização.
2. Configure os blocos: nome, duração em minutos e ritmo (min:seg, ex: 5:49).
3. Alertas sonoros + voz 30s antes de cada troca de bloco.
4. Monitor fala: "É só manter" · "Acelera!" · "Vamos diminuir!" a cada 20s.
5. Sem GPS? Use o simulador de ritmo (slider) durante o treino.
